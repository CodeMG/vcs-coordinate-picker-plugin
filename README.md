# coordinate_picker

> Part of the [VC Map Project](https://github.com/virtualcitySYSTEMS/map-ui)

## Description
This plugin provides a simple coordinate picker tool.
Once installed, you can find the tool here:
<img width="550" height="46" alt="image" src="https://github.com/user-attachments/assets/3fde43eb-8837-441f-84c7-e975a3710477" />

When toggled, click on a point on the map. A Window should open which shows you the coordinates of the point:
<img width="787" height="571" alt="image" src="https://github.com/user-attachments/assets/96165997-6cb5-4ea0-a94e-ed3d1e93d244" />
